Documentation for My Metal Bands API

This document will outline the technicalities of My Metal Bands site.

request url response
GET request /bands return array of all bands
GET request /bands/:bandName return band by name
GET request /genres return array of all bands
GET request /genres/:genre return information about a specific band genre
GET request /labels/:labelInfo return information about specific label
GET request /users return array of all users
POST request /users register a new user to the web application
PUT request //users/:username Update the user information
POST request /users/:username/bands/:bandID Add a band to a specific user's profile
DELETE request /users/:username Unregister a user from the database
DELETE request /users/:username/bands/:bandID Delete a band from the users favorites
POST request /bands Add a new band
DELETE request /bands/:bandname Delete a band
