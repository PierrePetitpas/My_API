Documentation for My Metal Bands API

This document will outline the technicalities of My Metal Bands site.

request url response
GET request /bands return array of all bands
GET request /bands/:bandName return band by name
GET request /genres/:genre return information about a specific band genre
GET request /bands/:bandInfo return information about specific band
POST request /register register to the web application
PUT request /users/:ID/:infoUpdate/:newValue Update the user username information
POST request /users/:ID/favorites/:newFavorite Add a band to a specific user's profile
DELETE request /users/:id/unregister Unregister a user from the database
DELETE request /users/:ID/favorites/:deleteFavorite Delete a band from the users favorites
